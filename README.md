# Factorial 🔢

Este repositorio contiene implementaciones y estudios algorítmicos sobre el cálculo de factoriales, enfocándose en la eficiencia de procesamiento y la gestión de grandes volúmenes de datos numéricos.

## 🚀 Propósito del Proyecto

El objetivo es explorar diferentes metodologías para el cálculo de factoriales, desde enfoques recursivos básicos hasta algoritmos optimizados para manejar números de gran magnitud, evitando el desbordamiento de memoria y maximizando la velocidad de respuesta.

## 📁 Estructura del Repositorio

Siguiendo el estándar de organización para una gestión eficiente de código:

* **src/**: Scripts y módulos (incluyendo implementaciones en Visual Basic 6) para el cálculo de factoriales.
* **docs/**: Documentación técnica sobre la complejidad computacional y análisis de rendimiento.
* **results/**: Tablas de resultados, tiempos de ejecución y benchmarks de los diferentes algoritmos.

## 🛠️ Tecnologías Aplicadas

* **Optimización Algorítmica**: Implementación de técnicas para reducir la carga computacional en operaciones intensivas.
* **Desarrollo en VB6**: Uso de tipos de datos avanzados y lógica estructurada para cálculos matemáticos precisos.
* **Integración de Sistemas**: Preparación de módulos de cálculo para ser utilizados en proyectos más amplios de análisis numérico.
